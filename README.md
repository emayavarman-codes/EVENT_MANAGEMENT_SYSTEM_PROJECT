 Event Management System (PHP + MySQL + XAMPP)
1. Project Overview

A web-based Event Management System designed to create, manage, and organize events efficiently. The system allows admins and users to handle event details, registrations, and attendee management using a simple and user-friendly interface.

2. Features

Create, update, and manage events

User registration & login system

Event booking/registration

Admin dashboard

MySQL database integration

Responsive UI using HTML, CSS, and Bootstrap

3. Technologies Used

PHP – Backend scripting

MySQL – Database

XAMPP – Local development server

HTML, CSS – Frontend

phpMyAdmin – Database management
